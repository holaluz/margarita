# Getting Started

## Installing Margarita locally

Clone the repo and install node dependencies:

```bash
$ npm install
```

Then you can run several commands such as

```bash
# run storybook locally
$ npm start

# run unit tests
$ npm test

# run unit tests with watch mode enabled
$ npm run test:watch
```
